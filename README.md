# SDE Interview Preparation Repository

A comprehensive collection of Data Structures, Algorithms, and System Design resources for Software Development Engineer (SDE) interviews. This repository covers everything from fundamental data structures to advanced problem-solving techniques used in top tech companies.

---

## Repository Structure

### **1. Data Structures & Algorithms** (`dataStructures/`)
Core implementations and algorithms:
- **Graph Algorithms**: Dijkstra, Bellman-Ford, Kruskal, Prim's MST
- **Dynamic Programming**: Knapsack problems
- **Greedy Algorithms**: Interval scheduling, optimization problems
- **Sorting**: Bubble, Insertion, Merge, Quick Sort (Java & Python)
- **Trees**: DFS, traversal algorithms
- **Heaps**: Min/Max heap implementations
- **Advanced**: KD-Tree, Skip List

### **2. LeetCode Problems** (`leetCode/`)
- **Blind 75**: Complete set organized by topics (Arrays, DP, Graphs, Trees, etc.)
- **Problem Solutions**: 50+ solved problems with implementations
- **Topic Coverage**: Arrays, Strings, Linked Lists, Trees, Graphs, DP, Backtracking, Heaps, Stacks, Greedy

### **3. Striver's SDE Sheet** (`striver/`)
Structured Java implementations following Striver's roadmap:
- **Arrays**: Easy, Medium, Hard problems (Two Sum, Merge Intervals, Next Permutation, etc.)
- **Sorting Algorithms**: Complete implementations with explanations
- **Basics**: Hashing, Recursion, Math, Patterns
- **Tree Problems**: Comprehensive tree traversal and manipulation

### **4. Creative Problem Solving** (`sitareCPS/`)
Course materials and practice problems:
- Array & String manipulation
- Recursion & Dynamic Programming
- Graph problems (BFS/DFS)
- Stack & Queue implementations
- Maze problems, Phone Number Mnemonics
- Mock interview problems

### **5. System Design** (`systemDesign/`)
System design fundamentals and patterns for scalable systems.

### **6. Additional Topics**
- **Puzzles** (`puzzles/`): Logical and algorithmic puzzles
- **Computer Networks** (`computerNetwork/`): Networking concepts
- **DBMS** (`dbms/`): Database management and SQL queries
- **Notes** (`notes/`): Quick reference guides, tricks, and learning resources
